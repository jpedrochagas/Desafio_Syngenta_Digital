Neste desafio da Syngenta Digital, foi necessário utilizar tecnicas avançadas de programação para desvendar a quantidade de pixels verdes presentes em uma imagem e desvendar o mistério de uma mensagem secreta escondida na foto. 
Uma imagem nada mais é do que uma matriz em três dimensões, compostas pelas cores RED, BLUE E GREEN "RGB". 
Estes valores são agrupados em cada pixel para que se obtenha a tonalidade de cor necessária.
Utilizei meus conhecimentos em python para fazer um programa que criava percorria uma matriz e montava um array dela, contando a quantidade de "cores em RGB" que possuia essa imagem, mostrando assim o valor da quantidade pontos verdes. 
Meu código comentado pode ser vizualizado no arquivo main.py
Minha explicação com os prints estão no arquivo .PDF


Instruções do desafio:

1. Conte a quantidade de pixels verdes.
2. A imagem possui uma mensagem escondida, descubra qual a mensagem.
